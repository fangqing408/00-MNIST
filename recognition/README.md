# 00-MNIST

## 1 >> 依赖环境

本项目因为是作者手写的 CNN，所以没有用到 GPU，因为本身手写字识别就是一个很简单的分类问题。

主要用到了一些简单的 Pytorch 库函数，来代替手写的 for 循环，不让代码看着过于的冗杂。

- 硬件配置
```
能开机的电脑
```

- 项目环境
```
numpy                     1.26.3
python                    3.9.21
tqdm  
```
